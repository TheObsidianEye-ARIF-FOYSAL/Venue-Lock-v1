# VenueLock App Icon — Design Prompt

Use this prompt with an image generator (Midjourney, DALL·E, Fable, etc.) whenever
the app icon needs to be regenerated or reimagined.

## Prompt

> A modern, minimal app icon for "VenueLock" — an event venue and seat-booking
> app whose tagline is "Lock the headcount. Unlock smooth entry." The icon
> should fuse the concept of a **padlock** with a **seat / venue map pin**:
> a single rounded padlock shape where the shackle doubles as a subtle seat-row
> arc or pin-drop silhouette. Flat vector illustration style, no photorealism,
> no gradients busier than a simple two-tone diagonal blend, no text or
> lettering. Bold, geometric, instantly legible at 48×48 px. Centered
> composition with generous padding (safe-zone) for adaptive icon masking.

## Style requirements

- **Primary color**: deep indigo `#3730A3` (the app's brand color), optionally
  blended with a slightly lighter indigo/violet for a subtle diagonal gradient.
- **Accent**: white or near-white for the lock/seat silhouette, for maximum
  contrast against the indigo field.
- **Shape language**: rounded corners, soft geometric forms — matches the
  app's existing rounded-rect cards and glass-morphism UI.
- **Single focal glyph**: one lock-plus-seat mark, not a busy scene. Must read
  clearly as a tiny status-bar or home-screen icon.
- **Background**: solid or two-tone gradient fill, edge-to-edge (no
  transparent corners) — required for Android adaptive icons and iOS icons.
- **No text, no realistic photography, no drop shadows besides a very subtle
  one for depth.**

## Deliverables

- 1024×1024 master PNG/SVG (square, full-bleed background)
- Android adaptive icon split: separate foreground glyph layer + background
  color/gradient layer
- Exported at standard mipmap sizes: 48, 72, 96, 144, 192 px (mdpi → xxxhdpi)
